The VIE Type system is capable of generating Backbone Forms schemas. This makes it easy to build forms to manage your VIE entities.

This example loads the Schema.org BarOrPub type into VIE and builds a form for it. The form schema contains everything schema.org knows about pubs, but we customize the fields to show only some interesting ones.
